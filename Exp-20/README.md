
##  Learning Outcomes

- **Full-Stack Development**: Build a complete web application with Flask REST API and React frontend
- **CORS & API Integration**: Configure Cross-Origin Resource Sharing and implement proper API communication
- **Docker Containerization**: Create multi-stage Dockerfiles and orchestrate microservices using Docker Compose
- **Frontend-Backend Validation Alignment**: Implement client-side form validation that mirrors backend requirements
- **CI/CD Automation**: Set up GitHub Actions workflows to automatically test both Python and Node.js services
